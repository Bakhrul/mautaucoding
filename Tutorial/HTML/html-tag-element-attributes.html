HTML : Tag, Element, dan Attribute pada HTML

  Setelah pada tutorial sebelumnya, kita sudah mengenal apa itu HTML?? , penulisan kode sederhana HTML, memilih teks editor sesuai selera masing - masing, dan menjalankan file HTML pada web browser. Pada kesempatan kali ini kita akan belajar apa yang dimaksud dengan Tag, Element, Attribute pada HTML yang merupakan dasar bagi sebuah website.

Pengertian Tag Pada HTML

  Tag adalah kode khusus untuk memberitahu web browser fungsi sebuah text, baik text tersebut berupa paragraf, list, link, dan sebagainya. Pada umumnya ada 2 jenis tag yakni tag pembuka dan tag penutup. Penulisan tag pembuka adalah <tag> sedangkan tag penutup </tag>.

Berikut ini beberapa contoh Tag HTML dasar :

1. Tag Heading / Judul

2. Tag Paragraph / kontent

3. Tag Line Break / ganti baris

4. Tag List / daftar

Mungkin ada yang memiliki tugas sekolah atau keperluan lain, Berikut ini beberapa kumpulan tag beserta fungsinya :
